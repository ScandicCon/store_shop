# 🛒 Online Shop (FastAPI)

Небольшой fullstack-проект интернет-магазина на FastAPI с простым фронтендом и системой авторизации.

---

## 🚀 Функционал

### Backend (FastAPI)

* Регистрация пользователей
* Авторизация (JWT)
* Получение текущего пользователя (`/users/me`)
* Работа с товарами:

  * Создание товара
  * Получение списка товаров
  * Получение товара по ID
* Загрузка изображений
* SQLite база данных

### Frontend (HTML + JS)

* Регистрация и логин
* Работа с JWT токеном
* Создание товаров
* Загрузка изображений
* Просмотр списка товаров
* Поиск товара по ID

## ⚙️ Установка и запуск

```bash
# клонирование
git clone <your-repo>
cd online_shop

# виртуальное окружение
python -m venv venv

# активация (Windows)
venv\Scripts\activate

# установка зависимостей
pip install -r requirements.txt

# запуск
python run.py
```

Или:

```bash
uvicorn app.main:app --reload
```

## 📡 API

### Auth

* `POST /register` — регистрация
* `POST /token` — логин

### Users

* `GET /users/me` — текущий пользователь

### Products

* `POST /product/append` — создать товар
* `GET /all/products` — список товаров
* `GET /product/{id}` — получить товар

### Files

* `POST /upload-image` — загрузка изображения

---

## 🖼️ Работа с изображениями
